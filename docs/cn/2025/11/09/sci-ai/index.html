科研搭配 AI合集

Tan Jay / 2025-11-09

视频介绍AI工具使用范式

  • How to write an exceptional literature review with AI

1. 查找文献:

  • https://www.bohrium.com/：用于找文献，可以设置所跟踪关注的学者们的最近成果。
  • https://www.semanticscholar.org/：大模型找文献。
  • https://scholar.google.com/：谷歌搜索。
  • https://arxiv.org/：未发表但是最新成果。
  • http://www.xueshu5688.com/：中文文献搜索。
  • https://www.proquest.com/：国外硕博毕业论文搜索。

2. 管理文献：

  • https://www.mendeley.com/：既可以找文献还可以管理文献。
  • https://www.zotero.org/：最推荐的管理文献神器。

3. 关联文献：

  • https://app.researchrabbit.ai/：给几个文献种子给你跟踪一系列文献，帮助厘清文献之间的脉络。
  • https://www.connectedpapers.com/：与上面一样，发现文献之间的关系，相关性，时间。
  • https://www.litmaps.com/：综上述功能都有，支持写文献综述。

4. 读文献：

  • https://docanalyzer.ai/app：使用较少，不同模型要自己买token，不如直接拿着文献跟自己喜欢的模型交流。
  • https://doc2x.noedgeai.com/：智能图文解析，基于Doc2X强大提取能力，理解文档与图片里每一个细节。

4. 写综述：

  • https://elicit.com/：写综述，写提纲，交流研究想法，导出文献摘要，…
  • https://consensus.app/：可以根据你提供的句子，为你推荐文献，并为你润色，来自大四师弟推荐。
  • https://scite.ai/：写文献综述的出名网站
  • https://www.paperdigest.org/review/：功能更全面的文献综述网站，自己感受，反正王炸级，需要有很好的阅读习惯，记录习惯，写作习惯才能衬得这把牛刀。

5. 本地科研agent搭建：

  • cursor + claude code + smithery

    • cursor AI 自动写代码，帮你解决本地终端安装 claude code
    • 用 claude code 给你本地生成秘书团
    • 给秘书团在 mcp 工具市场 smithery 找好工具
    • 于是秘书团就可以为你干活了
    • 参考小红书【秋芝】博主的《手把手打造私人AI秘书团！【小白教程】》，很喜欢她分析这篇《【AI失业潮】我们离大规模失业有多远？》。

    本人是按照秋芝如是说，搭建好的，博主教我真东西，才25岁呢！搭建出来的科研团队给我找文献的，读文献，写东西（目前测试了一下写日记）。找的文献很准确，阅读文献很详细，很贴近个人需求，缺点就是烧 token，换句话说就是费钱，最后开了一张虚拟 VISA 卡，改官方套餐 pro，这对我的阅读 2 篇 任务来说，还是不太够用，直接给我限制了，尬住。也许我应该调整思路。但确实能找，能读，能写，还说啥。接下来就是我与claude的磨合期，让他更懂我。

    6. 国外网站充值订阅问题

  • https://www.limaopay.com/：这是没去过国外的人的痛，挨个试遍很多终于找到众多信息中最靠谱的。

    心理历程大概是这样的：我订阅国外网站，没有境外卡，找代充，中间商赚差价不说，还有被封号的风险，于是就想拥有自己的卡，自己管理。试过很多方式，上述的狸猫网站做到了，给大家节约时间避坑。问题在于首次充值要100刀，但是是属于自己账号的钱，支持提现。

写在最后

  • https://www.zotero.org/：强推 zotero ！对于基本的文献保存，文献关联，文献阅读，文献翻译，文献总结，询问文献，文献综述，安装相应的插件，一应俱齐了。新手友好，集成高，性价比高。
    • 开发者 Polygononon 的三款插件包揽上述功能：Zotero All Pro系列近期功能更新：Zotero GPT/Reference/Style Pro
    • 很推荐 Polygononon 开发的 zotero 的 GhatGPT 插件的 pro 功能，支持从自己文献库一键上传到网页，如 https://chatgpt.com/，https://chat.deepseek.com/，https://gemini.google.com/app，https://claude.ai/等，然后让网页大模型读论文，分析论文，交流想法，写总结，不卡顿。

上述只是工具，还是试过之后选一两件称手趁手的工具，然后能沉浸学习就好了。有更好的科研方式欢迎来信分享。
